# Brand assets

- `mark.svg` — square app icon / favicon source (512×512)
- `logo.svg` — wordmark lockup for headers, docs, and slide decks
- `design-tokens.json` — the color palette and type scale used across the
  frontend (`tailwind.config.js`), the pitch deck in `/presentation`, and
  these assets — kept in one place so they don't drift apart

## The signature motif

Every surface — the web app, the admin dashboard, this logo, the slide deck —
reuses the same element: a rotated, outlined **status stamp**. It's not
decoration; it always carries the real state of something (an issue's review
status in the app, a visual echo of that idea in the brand mark). If you add
a new surface, reuse this motif rather than introducing a new one.
